◆ load_obj_and_bones()

Mesh Engine::Assets::load_obj_and_bones ( std::filesystem::path  path)

loads an .obj file and a .bones file containing bones weight and indices

A .bones contains the following format: vertex_index1 bone_index1|bone_weight1,bone_index2|bone_weight2,bone_indexN|bone_weightN,
vertex_index2 ... The vertex index corresponds with the .obj vertices indices, except that they start at 0 and not 1, this also applies to the bone_indices

Parameters
path_without_exta path pointing to two files with diferent extension but with the same name, i.e. path.obj and path.bones
